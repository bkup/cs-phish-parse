# cs-phish-parse
Parses TSV output data generated by the Cobalt Strike Phishing module.  Will calculate statistics on a per campaign basis for total emails sent, total unique targets, unique targets who clicked, unique click rate, and total clicks.

To generate TSV data: On the Cobalt Strike console navigate to Reports > Export Data.  Browse to the path it identifies to access to the TSV data.  The Spearphishes.tsv and Weblog.tsv will be necessary files.

-w: Weblog.tsv exported from CS

-s: Spearphishes.tsv exported from CS

-e: Text file containing email addresses to exclude; one-per-line

-x: Text file contain email subjects to exclude; one-per-line
